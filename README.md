# module-boot
自定义模块与组件的基础工程,来自[generator-rc](https://github.com/react-component/generator-rc)

## 使用

```
$ git clone https://github.com/LANIF-UI/module-boot.git
$ cd module-boot
$ yarn
$ yarn start
```

### 文件结构

```
- examples
 - index.html
 - index.js
- src
 - Component.js
 - index.js
- index.js
- tests
  - index.js
- package.json
```
